# AI Gesture Magic Camera | AI 国风玄术手势相机

一个基于电脑摄像头的实时互动特效程序。项目使用 MediaPipe 识别手部关键点，用 Pygame 渲染国风法术视觉，并结合 NumPy 粒子系统、四季背景、五行主题、电影感后处理和程序化音效，适合录屏展示、短视频特效演示和互动作品集。

## 功能特色

- 实时摄像头手势识别，支持单手和双手动作。
- 五行主题：木、火、土、金、水，可用 `1-5` 切换。
- 四季系统：春花、夏水光、长夏、秋叶、冬雪，每个季节有独立背景、粒子和环境声。
- 电影感画面：暗角、噪点、残影、光晕、屏幕闪白、震动和加色混合发光。
- 国风招式：八卦阵、剑指、食指雷法、道指休门阵、双手合十禅境、太极旋涡、掌中生莲、火焰、冰晶、龙游云海等。
- 程序化音效：无需额外音频文件，运行时生成四季环境声和动作触发音。
- GitHub 友好：已提供 `.gitignore`，会忽略缓存、本地 IDE 配置和旧资源。

## 环境要求

- Windows 10 / 11 推荐。
- Python 3.10、3.11 或 3.12。
- 可用摄像头。
- 建议使用干净虚拟环境，避免 MediaPipe、OpenCV、NumPy 版本冲突。

## 安装

```bash
pip install -r requirements.txt
```

如果安装 MediaPipe 失败，建议新建虚拟环境后重新安装：

```bash
python -m venv .venv
.venv\Scripts\activate
pip install --upgrade pip
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

程序启动后会自动打开摄像头。把手放进画面即可触发效果。

## 常见问题

### 摄像头打不开

检查摄像头是否被会议软件占用；确认系统已授权摄像头权限。也可以在 `config.py` 中修改 `CAMERA_INDEX`，例如从 `0` 改成 `1`。

### 画面卡顿

按 `-` 减少粒子数量，或在 `config.py` 中把 `LOW_QUALITY` 改为 `True`。如果仍然卡顿，可以降低摄像头采集分辨率。

### 手势识别不稳定

尽量保证光线充足，手掌完整进入画面。双手合十这类遮挡动作容易被识别成一只手，程序已加入时间序列补偿和角度限制，但实际效果仍受摄像头画质、距离和光线影响。

### 没有声音

请检查系统音量和默认播放设备。程序音效由 Pygame Mixer 生成，如果当前环境没有可用音频设备，声音系统会自动关闭，不会影响画面运行。

### MediaPipe 安装报错

本项目固定使用兼容版本：

```text
numpy<2
opencv-contrib-python<4.12
mediapipe==0.10.21
pygame>=2.5
```

如果本机装过其他版本的 MediaPipe / OpenCV，建议新建虚拟环境后重新安装。

## 说明

本项目偏向互动视觉演示，不是严格手势识别产品。不同摄像头、光线、手部距离都会影响识别效果。建议在明亮环境、纯净背景下使用。
